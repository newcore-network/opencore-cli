# @open-core/cli

Official command-line tool for the OpenCore Framework.

## Installation

```bash
npm install -g @open-core/cli
# or
pnpm add -g @open-core/cli
```

## Usage

```bash
opencore --help
```

For full documentation, visit: https://github.com/newcore-network/opencore-cli
