## [0.1.0] - 2024-12-06

### Added
- Initial release of OpenCore CLI
- `init` command for project scaffolding
- `create feature` command for generating features
- `create resource` command for generating resources
- `build` command with animated progress
- `dev` command with hot-reload
- `doctor` command for health checks
- `clone` command for official templates
- Beautiful terminal UI with Charmbracelet
- Interactive prompts with huh
- Animated spinners and progress bars
- TypeScript config support (opencore.config.ts)
- NPM wrapper package for easy installation
- GitHub Actions for CI/CD
- Cross-platform binaries (Windows, macOS, Linux)
- Embedded templates for project generation
- File watching with debouncing
